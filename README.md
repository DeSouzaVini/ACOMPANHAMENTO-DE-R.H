# Dashboard de RH - Power BI

Projeto desenvolvido em Power BI para acompanhamento de indicadores de Recursos Humanos.

## Objetivo

Analisar dados salariais e indicadores estratégicos de RH através de dashboards interativos.

## Ferramentas ultilizadas

- Power BI
- Power Query
- DAX
- Excel

## Arquivos do Projeto

- `ACOMPANHAMENTO DE R.H.pbix` → Dashboard Power BI
- `SALARIOS.xlsx` → Base de dados utilizada
- `Dashboard.png` → Imagem do dashboard

## Preview do Dashboard

![Dashboard](Dashboard.png)

## Indicadores Analisados

- Média Salarial
- Distribuição de Salários
- Indicadores de RH
- Comparativos
- Métricas estratégicas

## Como visualizar

1. Baixe o arquivo `.pbix`
2. Abra no Power BI Desktop
